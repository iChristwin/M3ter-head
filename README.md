# M3ter-head

Unique svg avatars for the M3tering Protocol, based on the sketch library [Bottts](https://bottts.com/) designed by [Pablo Stanley](https://twitter.com/pablostanley). Inspired and adapted from [magnusson](https://github.com/magnusson/bottts-maker)'s [Bottts Maker ðŸ¤–](https://github.com/magnusson/bottts-maker).


## Usage

to install the m3ter-head component package run: `npm install m3ter-head --save`


Import the M3ter component and put it render within your code

```jsx
import * as React from 'react'
import Bot from 'm3ter-head'

export default class MyComponent extends React.Component {
  render () {
    return 
      <div>
        Your M3ter:
        <M3ter seed={m3ter_id_string} />
      </div>
  }
}
```
